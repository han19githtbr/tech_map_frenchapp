04 · Visão Geral

Sobre a Arquitetura

🎨

Frontend

Construído com React + Next.js 15, com animações via Framer Motion e estilização utilitária com TailwindCSS 4. Gráficos com Recharts, PWA via next-pwa e Socket.io para real-time.

⚙️

Backend

Servidor Node.js com Next.js API Routes. Autenticação via NextAuth.js com OAuth Google. Banco de dados MongoDB acessado por Mongoose e Prisma.

🌐

Integrações

YouTube API para vídeos educativos, Freesound para efeitos sonoros, Cloudinary para imagens e Google OAuth para autenticação.

🚀

DevOps

Deploy na Vercel com CI/CD automático e edge functions. Docker para containerização e consistência entre ambientes.

Principais Integrações

  • ✓Frontend ↔ Backend — HTTP (Fetch API) + WebSocket (Socket.io)
  • ✓Backend ↔ Database — Mongoose ODM + Prisma ORM
  • ✓Backend ↔ APIs Externas — Axios / Node-fetch
  • ✓Autenticação — NextAuth.js com OAuth Google
  • ✓Real-time — Socket.io para atualizações ao vivo
  • ✓Imagens — Cloudinary CDN com otimização automática
